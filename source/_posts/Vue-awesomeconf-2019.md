Attended Vue Awesome Conf 2019 — a community conference focused on Vue.js. Here are my notes and highlights from the event.

## Key themes

The conference covered the Vue.js ecosystem broadly, with talks on:

- Vue CLI 3 and the new plugin system
- Composition API (early preview)
- Testing Vue components
- Vue + TypeScript patterns
- Performance optimisation techniques
