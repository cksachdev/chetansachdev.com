Logi Capture stopped working for many users on macOS Catalina and Mojave after newer versions dropped support. The fix is to install version 2.0.200 specifically, which prompts for the required permissions correctly.

## Steps

**Step 1: Uninstall the current version of Logi Capture**

Remove it from your Applications folder before installing the older version.

**Step 2: Download version 2.0.200**

Direct download link:
[Capture_2.0.200.zip](https://download01.logi.com/web/ftp/pub/techsupport/capture/Capture_2.0.200.zip)

Alternatively, find it on the [Logitech support page](https://support.logi.com/hc/en-us/articles/360025141614).

**Step 3: Install and launch**

Install the downloaded version. When you launch it, it will prompt for the necessary permissions (Camera, Microphone, Screen Recording). Grant them.

Logi Capture should now work normally on Catalina or Mojave.
